# GroupHome

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Value2021/GroupHome.git
   ```

2. Navigate into the project directory:
   ```bash
   cd GroupHome
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm start
   ```

## GitHub Pages Deployment

To deploy your site using GitHub Pages:
1. Make sure your code is pushed to the `main` branch.
2. Go to your GitHub repository settings.
3. Scroll down to the "GitHub Pages" section.
4. Select the `main` branch as your source and save.
5. Your site will be published at `https://Value2021.github.io/GroupHome`.  

Enjoy your web application!
